## Highlight Projects

### MyAnimeList x Spotify Integration
Because nobody should have to manually search for *"Again - YUI"* on Spotify after finishing FMA Brotherhood.

Built a tool that reads your completed anime from **MyAnimeList**, scrapes the theme songs, fuzzy-matches them against the **Spotify catalog**, and auto-generates playlists — all via OAuth. Features a custom normalization engine to handle the chaos of inconsistent song/artist naming across sources.

> `OAuth` · `Spotify API` · `MAL API` · `Web Scraping` · `Fuzzy Matching` · `Normalization Engine`

---

### ITSM x Endpoint Management Integrations
Enterprise tools are powerful. Enterprise tools that *talk to each other* are dangerous (in a good way).

Built custom bi-directional integrations bridging the gap between security/endpoint platforms and ITSM systems:
- **Rapid7 → ServiceNow** — Vulnerability data flowing straight into tickets
- **BigFix → ManageEngine** — Endpoint data synced for smarter asset management
- **Tanium → ServiceNow** — Real-time endpoint telemetry surfaced in ITSM workflows

> `REST APIs` · `ServiceNow` · `Tanium` · `Rapid7` · `BigFix` · `ManageEngine`

---

### Custom NagiosXI Monitoring Scripts
When the built-in checks aren't enough, you write your own. Built multi-system monitoring scripts for **NagiosXI** that extended out-of-the-box coverage for environments where the standard plugins just didn't cut it.

> `NagiosXI` · `Python` · `Bash` · `CheckMK` · `Multi-system Monitoring`

---

### CI/CD Pipelines for Unit Testing
Automated test pipelines so the code tells you when *it* broke — not your end users. Built and maintained pipelines across **GitHub Actions**, **GitLab CI**, and **Azure DevOps** to enforce quality gates and keep deployments boring (the good kind of boring).

> `GitHub Actions` · `GitLab CI` · `Azure DevOps` · `Unit Testing` · `Automated Pipelines`
